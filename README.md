# Vishal Gupta's Portfolio

A modern, responsive portfolio website showcasing my AI projects, skills, and experience in technology, leadership, and innovation.

## 🚀 Live Portfolio

The portfolio is deployed and live at: [VishalPortfolio.in](https://vishalportfolio.in)

## 📋 Features

- **Hero Section**: Eye-catching introduction with call-to-action buttons
- **About Me**: Personal background and key achievements
- **Skills**: Comprehensive list of technical and soft skills
- **Projects**: Showcase of completed projects with descriptions and tech stacks
- **Contact Section**: Easy ways to get in touch
- **Responsive Design**: Fully responsive on mobile, tablet, and desktop
- **Modern UI**: Sleek dark theme with smooth animations

## 🛠️ Tech Stack

- **Frontend**: React 18
- **Styling**: Tailwind CSS
- **Build Tool**: Vite
- **Icons**: Lucide React

## 📦 Installation & Setup

### Prerequisites
- Node.js (v16 or higher)
- npm or yarn

### Installation Steps

1. Clone the repository:
```bash
git clone https://github.com/codervishal108/VishalPortfolio.in.git
cd VishalPortfolio.in
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

The portfolio will open at `http://localhost:3000`

## 📝 Build & Deployment

### Build for Production:
```bash
npm run build
```

### Preview Production Build:
```bash
npm run preview
```

## 🎨 Customization

### Update Personal Information
Edit `src/components/Portfolio.jsx` to update:
- Name and title
- About section
- Skills list
- Projects information
- Contact details

### Modify Styling
- Update colors in `tailwind.config.js`
- Modify global styles in `src/index.css`
- Adjust component styles in `src/components/Portfolio.jsx`

## 🌐 Deployment Options
